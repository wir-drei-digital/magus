# Multiplayer

Multiplayer ermöglicht es mehreren Personen, gleichzeitig an derselben Unterhaltung teilzunehmen. Du kannst Kolleginnen und Kollegen einladen, verschiedene Rollen vergeben und sehen, wer gerade tippt.

## Multiplayer aktivieren

1. Öffne eine Unterhaltung.
2. Klicke auf das Multiplayer-Symbol im Unterhaltungs-Header (zwei Personen).
3. Schalte Multiplayer **ein**.

Sobald Multiplayer aktiviert ist, können Teilnehmer eingeladen werden.

## Teilnehmer einladen

1. Öffne bei aktiviertem Multiplayer das Teilnehmer-Panel.
2. Gib die E-Mail-Adresse der Person ein, die du einladen möchtest.
3. Wähle ihre Rolle: **Editor** oder **Observer** (siehe unten).
4. Klicke auf **Einladen**.

Die Person erhält eine Einladung. Hat sie bereits ein Magus-Konto, erscheint die Unterhaltung sofort in ihrer Seitenleiste. Andernfalls wird sie aufgefordert, ein Konto zu erstellen.

## Teilnehmer-Rollen

**Editor**-Teilnehmer können Nachrichten an die Unterhaltung senden, genau wie du. Nutze diese Rolle für Personen, mit denen du aktiv zusammenarbeiten möchtest.

**Observer**-Teilnehmer können alle Nachrichten lesen, aber keine senden. Nutze diese Rolle für Personen, die mitverfolgen sollen, ohne selbst beizutragen, zum Beispiel eine Führungskraft, die eine Unterhaltung überprüft.

## Live-Tipp-Anzeige

Wenn ein anderer Teilnehmer gerade eine Nachricht verfasst, erscheint unten in der Unterhaltung eine Tipp-Anzeige. Du siehst seinen Namen und eine pulsierende Animation. Diese aktualisiert sich in Echtzeit, sodass du immer weißt, ob jemand anderes gleich etwas sendet.

## Rolle eines Teilnehmers ändern

1. Öffne das Teilnehmer-Panel.
2. Suche den Teilnehmer, den du aktualisieren möchtest.
3. Klicke auf den Rollen-Selektor neben seinem Namen.
4. Wähle die neue Rolle.

Die Änderung tritt sofort in Kraft.

## Teilnehmer entfernen

1. Öffne das Teilnehmer-Panel.
2. Suche den Teilnehmer, den du entfernen möchtest.
3. Klicke auf das Entfernen-Symbol (X) neben seinem Namen.
4. Bestätige die Entfernung.

Entfernte Teilnehmer verlieren sofort den Zugriff auf die Unterhaltung, und diese verschwindet aus ihrer Seitenleiste.

## Multiplayer deaktivieren

Um Multiplayer auszuschalten und eine Unterhaltung wieder in den Einzelnutzer-Modus zu versetzen:

1. Öffne das Teilnehmer-Panel.
2. Entferne zuerst alle anderen Teilnehmer oder übertrage die Inhaberschaft, falls nötig.
3. Schalte Multiplayer **aus**.

Das Deaktivieren von Multiplayer beendet den Zugriff für alle Teilnehmer. Der Verlauf der Unterhaltung bleibt erhalten.
