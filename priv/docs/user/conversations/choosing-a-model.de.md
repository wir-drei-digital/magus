# Ein Modell auswählen

Magus bietet dir Zugang zu einer großen Auswahl an KI-Modellen verschiedener Anbieter. Jedes Modell hat seine eigenen Stärken, und die richtige Wahl für deine Aufgabe kann einen spürbaren Unterschied machen.

## Was Modelle unterscheidet

Nicht alle Modelle sind für denselben Zweck gebaut:

- **Unterhaltung und Texterstellung:** Allzweckmodelle sind schnell und vielseitig. Sie eignen sich für alltägliche Aufgaben wie das Verfassen von E-Mails, das Beantworten von Fragen und das Erklären von Konzepten.
- **Schlussfolgerungen:** Manche Modelle sind darauf ausgelegt, komplexe Probleme Schritt für Schritt durchzudenken. Sie sind in der Regel langsamer, aber genauer bei schwierigen Logik-, Mathematik- oder Analyseaufgaben.
- **Kreative Arbeit:** Bestimmte Modelle glänzen beim kreativen Schreiben, Brainstorming und bei offenen Aufgaben, bei denen Fantasie gefragt ist.
- **Modelle mit Suchfunktion:** Einige Modelle können als Teil ihrer Antwort im Web suchen – das ist hilfreich, wenn du aktuelle Informationen brauchst.
- **Tool-Nutzung:** Modelle, die Tools unterstützen, können Aktionen ausführen wie das Durchsuchen deines Speichers, das Ausführen von Code oder das Planen von Jobs.

## Fähigkeits-Badges

Im Modell-Selektor siehst du kleine Badges an jedem Modell, die seine besonderen Fähigkeiten anzeigen:

- **Search:** Das Modell kann aktuelle Informationen im Web nachschlagen.
- **Reasoning:** Das Modell unterstützt erweitertes Denken für komplexe Probleme.
- **Tools:** Das Modell kann Magus-Tools verwenden (Speicher, Websuche, Planung usw.).

Nicht alle Modelle haben alle Fähigkeiten. Wenn eine Funktion nach dem Wechsel des Modells ausgegraut ist, wird sie von diesem Modell möglicherweise nicht unterstützt.

## Das Modell in einer Unterhaltung wechseln

Du kannst das Modell jederzeit während einer Unterhaltung wechseln. Klicke auf den Modell-Selektor im Chat-Eingabebereich, um eine Liste der verfügbaren Modelle zu öffnen. Das gewählte Modell wird sofort für deine nächste Nachricht wirksam.

Ein Modellwechsel mitten in einer Unterhaltung ist problemlos möglich. Der Unterhaltungsverlauf bleibt erhalten, und das neue Modell hat den vollen Kontext aus allem, was bisher gesagt wurde.

## Standardmodelle festlegen

Du kannst in deinen Kontoeinstellungen Standardmodelle festlegen, damit du nicht jedes Mal ein Modell auswählen musst, wenn du eine neue Unterhaltung startest.

Magus unterstützt drei separate Standardmodelle:

- **Chat-Modell:** Wird für alle regulären Unterhaltungen, den Suchmodus und den Reasoning-Modus verwendet.
- **Bildmodell:** Wird verwendet, wenn du in den Bildgenerierungsmodus wechselst.
- **Videomodell:** Wird verwendet, wenn du in den Videogenerierungsmodus wechselst.

Um deine Standardmodelle zu ändern, gehe zu **Einstellungen** und suche den Abschnitt für Modelleinstellungen. Änderungen gelten für alle neuen Unterhaltungen ab diesem Zeitpunkt. Bestehende Unterhaltungen behalten das Modell, das sie bisher verwendet haben.

## Nutzungskosten

Verschiedene Modelle haben unterschiedliche Anbieterpreise. Leistungsstärkere oder spezialisierte Modelle kosten in der Regel mehr pro Nachricht. Die relativen Kosten eines Modells werden im Modell-Selektor angezeigt, damit du eine fundierte Entscheidung treffen kannst.

Die Wahl eines effizienteren Modells ist eine gute Möglichkeit, Pay-as-you-go-Ausgaben niedrig zu halten. Die kostenlose Testnutzung ist auf günstigere Modelle beschränkt, während Pay-as-you-go Premium-Modelle innerhalb deines monatlichen Ausgabenlimits freischaltet.
