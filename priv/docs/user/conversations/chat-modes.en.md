# Chat Modes

Magus offers several chat modes, each designed for a different kind of task. You can switch modes at any time using the mode selector in the chat input area.

## Chat

The default mode for everyday conversations. Use it for writing, coding, explaining concepts, brainstorming, and any general-purpose task.

**Best for:** Most tasks. If you're unsure which mode to use, start here.

## Image Generation

Switch to Image Generation mode to create images from text descriptions. Type a description of what you'd like to see and the AI will generate an image. You can iterate by refining your prompt or asking for variations.

The image model is separate from the chat model. You can change which image model to use in the model selector.

**Best for:** Creating illustrations, concept art, logos, photo-realistic scenes, or any visual content.

## Video Generation

Video Generation mode works like Image Generation, but produces short video clips from your description. Video generation takes longer than image generation, and you'll see a progress indicator while your video is being created.

**Best for:** Short animated clips, motion graphics, or video concepts.

## Switching modes

Click the mode selector in the chat input area (it shows the name of the current mode). A menu will appear with all available modes. Click the one you want to switch immediately.

Switching modes mid-conversation is fine. The AI will use the new mode for your next message and all subsequent messages in that conversation.
